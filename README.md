🎵 MusicStream - Angular Streaming Platform
MusicStream is a modern, responsive music streaming web application built with Angular and Angular Material. The project focuses on delivering a high-quality user experience with a sleek dark-themed UI, dynamic song discovery, and reactive state management.

🚀 **Features**
Dynamic Song Grid: Browse a curated catalog of tracks with high-quality cover art.

Responsive UI/UX: Fully optimized for Mobile, Tablet, and Desktop using Angular Material and Flexbox.

Theme Management: Integrated support for Dark and Light modes.

Reactive State: Powered by RxJS for seamless data handling and UI updates.

Mock API Integration: Uses JSON Server to simulate real-time REST API interactions.

🛠️ **Tech Stack**
Framework: Angular 17+

Language: TypeScript

Styling: SCSS, Angular Material, Flexbox

Data Handling: RxJS, HttpClient

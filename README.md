🎵 **MusicStream - Angular Music Streaming Application**
MusicStream is a modern, responsive music streaming web application built using the Angular framework and Angular Material. It features a clean, dark-themed UI that allows users to browse song catalogs, manage playlists, and experience seamless audio playback interaction.

🚀 **Key Features**
Dynamic Song Discovery: Browse a curated list of tracks with high-quality cover art and artist details.

Responsive UI: Fully optimized for Mobile, Tablet, and Desktop using Angular Material and Flexbox.

Theme Switching: Integrated Dark and Light mode support for an enhanced user experience.

Modular Architecture: Built with reusable components and services for high scalability.

Mock API Integration: Uses json-server to simulate real-time REST API data fetching via HttpClient.

Reactive State Management: Utilizes RxJS for smooth data handling and UI updates.

🛠️ **Tech Stack**
Framework: Angular 17+

Language: TypeScript

Styling: SCSS, Angular Material, Flexbox/Grid

Data Handling: RxJS, HttpClient

Backend Simulation: JSON Server
